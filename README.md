# Terminology (용어)
- Algorithm (알고리즘, 알고리듬)
- Data structure (자료 구조)
- Simple search (단순탐색)
- Binary search (이진탐색)
- Array (배열)
- Big O notation (빅오 표기법)
- Recursion (재귀)
- divide-and-conquer (분할정복)
- Recursive function (재귀함수)
- Pseudocode (의사코드)
- stack (스택)
- Call stack (호출스택)
- Push (푸시, 삽입)
- Pop (팝, 읽은 후 삭제)
- Quicksort (퀵정렬)
- Euclid's algorithm (or The Euclidean algorithm) (유클리드 알고리즘)
